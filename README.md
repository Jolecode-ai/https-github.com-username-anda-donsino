# Don'Sino

🎰 Multiplayer Poker, Gaple & Qiu Qiu — Bermain bersama teman!

## Cara Main

1. Buka link aplikasi
2. Login atau main sebagai Tamu
3. Pilih permainan (Poker / Gaple / Qiu Qiu)
4. Buat Room → bagikan kode 6 huruf ke teman
5. Teman masukkan kode → klik Ready
6. Host klik Mulai Game → Selamat bermain! 🎮

## Fitur

- 🃏 **Poker** Texas Hold'em (2-9 pemain)
- 🁣 **Gaple** Domino (2-4 pemain)
- 🎲 **Qiu Qiu** 99 Domino (2-6 pemain)
- 🔐 Login dengan Firebase (Email/Google)
- 🪙 Sistem beli chip virtual
- 💸 Pajak 2.5% dari kemenangan
- 🎨 UI premium dark casino

## Setup Lokal

```bash
npm install
npm start
```

Buka `http://localhost:3000`

## Deploy ke Render.com

1. Push repo ini ke GitHub
2. Buka [render.com](https://render.com) → New Web Service
3. Connect GitHub repo
4. Build Command: `npm install`
5. Start Command: `node server.js`
6. Done! 🚀
